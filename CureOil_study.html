The CureOil website was created for my systems analysis class in Westchester Community College. My assignment was to analyze how this website would be easy for users and also who will be purchasing the items. I've decided to create my own website instead of searching for a website to analyze because I wanted to practice more on creating shopping carts online. My reference to this project was lynda.com. I looked back on the exercises I've done to create an ecommerce website and I've applied that to the CureOil website.
